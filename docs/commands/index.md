# Command reference

The CLI is intentionally small and centered on the core image lifecycle.

## Commands

- `create` builds an image from a source tree.
- `check` validates structure, hashes, and tree layout.
- `ls` prints the tree representation of an image.
- `info` shows lightweight image metadata (header, size, version).
- `analyze` / `analyse` inspects image internals and validates layout and payloads.
- `extract` extracts logical files from an image to a directory.
