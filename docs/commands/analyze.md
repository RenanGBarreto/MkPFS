# `analyze` / `analyse`

Inspect image internals, validate layout, and verify payload hashes and signatures.

## Synopsis

```bash
mkpfs analyze --image ./output.ffpfs --print-tree
```

## Useful options

- `--print-tree`: print the parsed tree after the analysis.
- `--verbose`: enable additional diagnostic output.

## Notes

`analyse` is provided as a British English alias for `analyze`.
