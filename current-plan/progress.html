INODE_FLAG_SIGNED_EXTRA HIGH May be missing from consts.py. All inodes in signed builds require this 0xC flag. Missing → signed images differ. ⬜ Task 1/6
HEADER_DIGEST_OFFSET / SIZE HIGH Constants for signed header HMAC region (0x380 / 0x5A0) may not be in consts.py. Required for verify_signed_image_signatures. ⬜ Task 1/7
ZERO_EKPFS / ZERO_PFS_SEED MED Used for zero-credential signed image generation and verification. May be inline in pfs.py rather than named constants. ⬜ Task 1
INODE_S64_SIZE MED 0x310 constant for the header InodeBlockSig S64 structure. May be missing from consts.py (used in build_inode_block_sig_s64). ⬜ Task 1
check aliases ["verify"] MED Legacy build_cli registers aliases=["verify"] for check subcommand. New cli_mkpfs_main_parsers omits this. ⬜ Task 8
ndblock += 2 (signed build) MED Legacy skips 2 blocks before signed layout assignment (superroot + uroot dirs). Must verify this is present in new build_pfs. ⬜ Task 7
